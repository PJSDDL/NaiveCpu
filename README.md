三、机器周期
CPU有四个机器周期，在第一个周期，读取ROM内的指令，同时PC自增；在第二个周期，读取RAM内的数据或向RAM内写数据；在第三个周期，CPU对PC与X寄存器进行操作，第四个周期不做任何操作。
 
四、指令含义
一条指令十六位，设为XXXX_XXXX_XXXX_XXXX，各个位含义如下：
第15~12位，XXXX_XXXX_XXXX_XXXX，用于选择寄存器，含义如下：
数值	寄存器变化
0000	寄存器无变化
1000	X<=ALU-out 
0100	flag为1则PC<=imd，为0 PC不变
0010	MEM[imd] <=X
0001	X<=ALU-out  carry<=ALU-carry-out
otherwise	禁止使用

第11位，XXXX_XXXX_XXXX_XXXX，用于选择ALU输入，含义如下：
数值	ALU_in_A	含义
1	imd	指令后8位，即ROM中取出的数
0	mem[imd] 	将imd作为地址，从RAM中取数

第10~8位，XXXX_XXXX_XXXX_XXXX，用于控制ALU运算类型，含义如下：
数值	ALU-out
000	无进位加法
carry = 0
001	有进位加法
010	或非
011	内存数据或imd
100	X== RAM或ROM数值输出1,否则0
101	X
110	禁止使用
111	禁止使用

最后8位：XXXX_XXXX_XXXX_XXXX，立即数，imd
五、指令集
有意义的命令组合如下(D代表立即数)，其中[imd]代表以imd为地址，从RAM中取出的数：
0000000000000000		NOP
10001011DDDDDDDD		MOV X,imd
00100000DDDDDDDD		MOV[imd],X
10000011DDDDDDDD		MOV X,[imd]
10001100DDDDDDDD		EQ imd		（若X与imd相等，则X等于11111111）
10000100DDDDDDDD		EQ [imd]		（若X与[imd]相等，则X等于11111111）
01001000DDDDDDDD		JO imd		（若X等于11111111，则跳转至imd）
10001000DDDDDDDD		ADD imd
10000000DDDDDDDD		ADD [imd]
00011001DDDDDDDD		ADC imd
00010000DDDDDDDD		ADC [imd]

1000101100001010 ; MOV X, imd (imd = 00001010) ; X = 10 （被乘数）

0010000000000001 ; MOV [imd], X (imd = 00000001) ; MEM[1] = 乘数 = 16
;计算中间结果
1000001100000001 ; MOV X, [imd] (imd = 00000001) ; X = MEM[1] = 乘数 = 16
1000000000000010 ; ADD [imd] (imd = 00000010) ; X = X + MEM[2] （MEM[2]初始为0）
0010000000000010 ; MOV [imd], X (imd = 00000010) ; MEM[2] = 中间结果
;计算循环次数
1000101100000001 ; MOV X, imd (imd = 00000001) ; X = 1 （循环次数计数器初始值）
1000000000000011 ; ADD [imd] (imd = 00000011) ; X = X + MEM[3] （MEM[3]初始为0）
0010000000000011 ; MOV [imd], X (imd = 00000011) ; MEM[3] = 循环次数计数器
;循环开始：累加被乘数，直到计数器 = = 被乘数 
1000001100000000 ; MOV X, [imd] (imd = 00000000) ; X = MEM[0] = 被乘数 = 10
1000010000000011 ; EQ [imd] (imd = 00000011) ; if X == MEM[3]（循环次数） then flag = 1
0100100000010011 ; JO imd (imd = 00010011) ; if flag == 1 then PC = 00010011（跳出循环）
0000000000000000 ; NOP

0000000000000000 ; NOP
0000000000000000 ; NOP
